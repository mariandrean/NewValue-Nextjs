### Clone the Repository
```bash
git clone https://github.com/mariandrean/NewValue-Blog .
```

### Install the dependencies
```bash
npm install
```

### Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
